# Motivational Content Creator

An automated content tool that generates and publishes original motivational short-form videos to social media platforms.

## Features
- AI-generated motivational stories
- Professional voiceover narration
- Stock footage with text overlays
- Automated publishing to TikTok, Instagram Reels, and YouTube Shorts
