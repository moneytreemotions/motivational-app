# Privacy Policy - Motivational Content Creator

**Last updated: February 2026**

## Overview
Motivational Content Creator is a personal content automation tool that generates and publishes original motivational short-form video content to TikTok and other social media platforms.

## Data Collection
This application does not collect, store, or process any personal data from end users. It operates solely on the app owner's TikTok account to publish video content.

## Data Usage
No user data is collected, sold, shared, or transferred to third parties. The application only interacts with the app owner's authenticated social media accounts for the purpose of publishing content.

## Third-Party Services
This application uses TikTok's Content Posting API exclusively for publishing video content to the app owner's account. No data from TikTok users or viewers is accessed or stored.

## Data Retention
As no personal data is collected, there is no data retention policy applicable to end users.

## Children's Privacy
This application does not collect any data from children or any other users. All published content complies with TikTok's Community Guidelines.

## Changes to This Policy
Any changes to this privacy policy will be posted on this page with an updated revision date.

## Contact
For questions about this privacy policy, please open an issue on this repository.
